# 🏥 Medical Booking Web Application

## 📌 Description

This project is a full-stack web application designed for managing medical appointments. The main purpose of the application is to simplify the interaction between patients, doctors, and administrators by providing an easy-to-use online booking system.

Users can create accounts, browse available doctors, and schedule appointments based on availability.

---

## 👥 User Roles

The application is built around three main user roles:

* **Patient**
* **Doctor**
* **Admin**

Each role has access to specific functionalities within the platform.

---

## ⚙️ Features

### 🔹 Patient

* Register and log in
* View available doctors
* Filter doctors by specialization
* Book appointments
* View existing appointments
* Cancel appointments

### 🔹 Doctor

* Log in to a dedicated dashboard
* View incoming appointments
* Accept or reject appointment requests
* Manage availability schedule
* View patient details

### 🔹 Admin

* Manage users (patients and doctors)
* Add, update, or delete data
* Monitor appointments
* Maintain overall platform functionality

## 🚀 How to Run the Project

### 1. Clone the repository

```
git clone https://github.com/your-username/your-repo-name.git
```

### 2. Install dependencies

#### Backend

```
cd backend
npm install
```

#### Frontend

```
cd frontend
npm install
```

### 3. Run the application

#### Start backend

```
npm start
```

#### Start frontend

```
npm start
```
